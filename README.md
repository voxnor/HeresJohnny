## Meet Satisfactory's most famous Hog

As you may or may not know, Satisfactory (vanilla) includes a very small chance for certain Hog spawns to instead spawn __Johnny the Jetpack Hog__.

The __Here's Johnny__ mod is a very small mod that adds new spawn locations throughout the game world where __Johnny the Jetpack Hog__ is guaranteed to spawn. You can install the mod and simply play as normal - and you're very likely to bump into him. Or, check out the spoiler map down below for approximate locations. 

_Surviving your encounter and claiming his jetpack for yourself? That's on you._

## Features:

  - Adds 11 New spawn locations around the game world that always spawn __Johnny the Jetpack Hog__
    
    _Each __Johnny the Jetpack Hog__ location can only be defeated once per save file, and then will not return._

### Multiplayer and Dedicated Server
  - Lightly tested on Windows dedicated server and local IP multiplayer. Found no immediate issues and believe the mod should work correctly in both environment types. However, testing was not extensive. If you encounter issues or have questions, please reach out to me using the contact information listed below.
